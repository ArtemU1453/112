# Итоговый аудит проекта — индекс (Stage 10)

Итоговые отчёты проектного аудита ЕАСУР (подэтап 10.1, 10.10). Формируются при завершении проекта на
основании всех этапов, стандартов и решений (ADR/RFC).

- [Architecture Compliance Report](architecture-compliance-report.md) — соответствие реализации
  утверждённой архитектуре (SSOT, Architecture Freeze, трассируемость).
- [Project Completion Report](project-completion-report.md) — завершение проекта, итоги этапов 1–10.
- [Production Readiness Report](production-readiness-report.md) — итоговая готовность к промышленной
  эксплуатации и условия go-live.

Связанные документы: [Operations](../operations/README.md) · [ADR Index](../adr/README.md) ·
[RFC Index](../rfc/README.md) · [EAGF](../governance/Enterprise-Architecture-Governance-Framework.md).
