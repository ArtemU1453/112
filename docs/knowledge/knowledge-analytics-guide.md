# Knowledge Analytics Guide — ЕАСУР

- **Status:** APPROVED · Stage 13 · Реализует подэтап 13.10. Согласован с ADR-028, Knowledge Quality
  Standard, Analytics Governance Standard (Stage 8), KPI Governance Standard.

## 1. Назначение
Аналитическая подсистема базы знаний: использование документов, популярные запросы, устаревающие
документы, эффективность SOP, пробелы в базе знаний, рекомендации по актуализации.

## 2. Показатели и представления
| Область | Показатели |
|---------|-----------|
| Использование | Обращения к документам/SOP/правилам; активные/неиспользуемые единицы |
| Поиск | Популярные/безрезультатные запросы; качество результатов (полнота/точность) |
| Актуальность | Устаревающие документы (по сроку/изменению нормативов) |
| Эффективность SOP | Частота применения, отклонения на контрольных точках (совокупно, обезличенно) |
| Пробелы | Темы без покрытия; безрезультатные запросы как индикатор пробелов |
| Ознакомление | Прохождение обязательного ознакомления; просрочки |

## 3. Рекомендации по актуализации
На основе показателей формируются рекомендации: обновить устаревающее, восполнить пробелы, пересмотреть
малоэффективные SOP. Рекомендации совещательны; изменения проводятся через утверждённый процесс
(Knowledge Publication), архитектурные/процессные — через RFC→ARB→ADR.

## 4. Принципы
Обезличенность аналитики использования (приватность); опора на утверждённые версии; согласование метрик
с KPI Governance; воспроизводимость расчётов (Statistical Methodology, Stage 8).
