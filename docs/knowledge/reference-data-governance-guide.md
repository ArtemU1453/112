# Reference Data Governance Guide — ЕАСУР

- **Status:** APPROVED · Stage 13 · Реализует Reference Information Standard (13.5); согласован с
  Reference Data Management Standard (Stage 9), Master Data Governance, Canonical Identifier Standard.

## 1. Назначение
Руководство по централизованному управлению справочной информацией платформы знаний: классификаторы,
словари, коды, перечни, нормативные значения, отраслевые справочники.

## 2. Каталог справочников
| Класс | Примеры | Владелец |
|-------|---------|----------|
| Классификаторы | Типы происшествий/объектов/ресурсов | Домен реагирования |
| Словари | Термины/сокращения | Владельцы знаний |
| Коды | Кодовые перечни, статусы | Домен/платформа |
| Нормативные значения | Нормативы времени/покрытия | Нормативный орган |
| Отраслевые справочники | Ведомственные справочники | Владелец справочника |

## 3. Управление версиями
Каждый справочник версионируется; применяются только утверждённые версии; история неизменяема
(Knowledge Retention Standard). Изменения — через утверждённый процесс (Knowledge Publication).

## 4. Согласованность и каноничность
Идентификаторы согласованы с Canonical Identifier Standard (Stage 9); при пересечении с НСИ
межведомственного обмена — синхронизация с Reference Data Management Standard (Stage 9) как источником.
Ссылки правил/SOP/документов используют актуальные коды.

## 5. Качество
Полнота/непротиворечивость/актуальность контролируются (Knowledge Quality Standard); устаревающие
значения выявляются аналитикой и актуализируются.

## 6. Связи
Reference Information Standard · Reference Data Management Standard (Stage 9) · Master Data Governance ·
Canonical Identifier Standard · Knowledge Quality Standard.
