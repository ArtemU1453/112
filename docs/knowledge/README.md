# Знания, нормативы и поддержка решений — индекс (Stage 13)

Национальная платформа управления знаниями, нормативными документами, справочниками и правилами
поддержки решений (подэтапы 13.1–13.10, ADR-028, RFC-0010). **Единый источник (SSOT)** нормативной
информации; применяются только **утверждённые** версии; история изменений **неизменяема**; правила
имеют документированное основание; поддержка решений **объяснима и совещательна** (ИИ не решает за
человека). Бизнес-логика оперативных модулей не изменяется (изменения процессов — через RFC→ARB→ADR).
