# Digital Twin Operations Handbook — ЕАСУР

- **Status:** APPROVED · Stage 12 · Эксплуатация платформы двойника/моделирования. Согласован с
  ADR-026, Simulation Governance / Security Standards, Platform Operations Framework (Stage 3).

## 1. Назначение
Руководство по эксплуатации изолированного контура цифрового двойника и моделирования: развёртывание,
обновление модели, проведение прогонов/учений, наблюдаемость, безопасность, обслуживание. Контур
эксплуатируется **отдельно** от промышленного (вмешательство в прод запрещено).

## 2. Развёртывание и изоляция
Изолированный контур (собственные инфраструктура/кластер, self-hosted, AGSDS). Сетевой доступ к прод,
позволяющий влиять на него, отсутствует. Импорт справочной модели — только офлайн-снимком (Digital Twin
Governance).

## 3. Регламентные операции
| Операция | Действия |
|----------|----------|
| Обновление двойника | Импорт офлайн-снимка → построение новой `TwinVersion` → валидация → публикация |
| Запуск прогона/серии | Выбор версий сценария/двойника, параметры, seed → Run Orchestrator → сбор результатов |
| Проведение учения | По Exercise Management Guide; фиксация решений; отчёты |
| Эксперимент ИИ | По AI Experimentation Policy (ADR-027); только синтетика; без переноса в прод |
| Архивирование | Версии/результаты — в хранилище истории (ретенция, Exercise Data Governance) |

## 4. Наблюдаемость
Метрики/логи/трассы контура (по образцу Stage 3, но изолированно): состояние прогонов, ресурсы,
ошибки, целостность данных. Дашборды и алерты — в пределах контура.

## 5. Безопасность
Zero Trust, RBAC, секреты в Vault, аудит доступа (Simulation Security Standard). Синтетические данные
не смешиваются с реальными; экспорт — с маркировкой источника.

## 6. Обслуживание и восстановление
Резервное копирование состояний двойника/истории экспериментов; восстановление — независимо от прод;
контур можно пересоздать без влияния на промышленную систему.

## 7. Инварианты (обязательные проверки этапа)
Двойник изолирован; моделирование воспроизводимо; результаты сохраняются; экспериментальные модели ИИ
не влияют на прод; учебные сценарии версионируются.
