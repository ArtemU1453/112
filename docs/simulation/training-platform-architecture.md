# Training Platform Architecture — ЕАСУР

- **Status:** APPROVED · Stage 12 · Подчиняется Architecture.md, ADR-026, Training Evaluation /
  Scenario Versioning / Exercise Data Governance Standards. Изолированный контур.

## 1. Назначение
Учебная среда и национальная платформа учений (12.4, 12.8): обучение диспетчеров, руководителей,
подразделений; экзаменационные сценарии; командно-штабные и межслужебные тренировки; оценка и отчёты.

## 2. Компоненты
| Компонент | Функция |
|-----------|---------|
| Training Scenario Manager | Учебные/экзаменационные сценарии (версионированные) |
| Trainee Workspace | Учебные АРМ диспетчера/руководителя (изолированные от прод) |
| Multi-user Exercise Engine | Многопользовательские сценарии, взаимодействие служб |
| Injects Controller | Вводные/эскалации по временной шкале |
| Decision Recorder | Автоматическая фиксация решений/действий/таймингов |
| Evaluation Engine | Оценка действий по критериям (Training Evaluation Standard) |
| Debrief & Reporting | Разбор ошибок, итоговые отчёты, Lessons Learned |

## 3. Роли и режимы
Обучаемые (диспетчеры/руководители/подразделения), инструкторы (управление вводными/оценка),
наблюдатели. Режимы: индивидуальное обучение, экзамен, командно-штабные учения (несколько служб).

## 4. Оценка и фиксация
Действия фиксируются автоматически и привязываются к версии сценария и обучаемому (неизменяемо).
Оценка — по критериям своевременности/корректности/полноты/взаимодействия; ИИ — совещательный.
Результаты хранятся (Exercise Data Governance / Simulation Analytics).

## 5. Изоляция
Учебная среда — в контуре двойника (ADR-026); **учебные и реальные инциденты не смешиваются**
(маркировка `origin=simulation`); данные синтетические. Нет связи с боевой диспетчеризацией.
