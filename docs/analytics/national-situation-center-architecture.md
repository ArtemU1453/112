# National Situation Center Architecture — ЕАСУР

- **Status:** APPROVED · Stage 8 · Подчиняется Analytics/BI Architecture, GIS Architecture (Stage 6).

## Назначение (8.8)
Единый ситуационный центр национального уровня: карта страны, оперативная обстановка, активные
происшествия, крупные ЧС, распределение сил и средств, ключевые показатели, журнал критических
событий.

## Состав
- **Карта страны** — оперативная карта (Stage 6, ADR-015 self-hosted слои) в масштабе страны/областей.
- **Оперативная обстановка** — активные инциденты и крупные ЧС (агрегаты по регионам).
- **Распределение сил и средств** — сводка доступности/занятости (Resource Availability, Stage 6).
- **Ключевые показатели** — национальные KPI (KPI Platform, 8.3) и дашборды (ADR-020).
- **Журнал критических событий** — лента критических происшествий/эскалаций.

## Данные и обновление
Real-time из оперативного контура (Kafka→realtime) + агрегаты из DWH. Обновление в реальном
времени; исторический контекст из хранилища. Единая BI/DWH-основа (без дублирования).

## Безопасность
Доступ — высший уровень управления (RBAC, BI Security Standard); данные защищены; аудит доступа.
Суверенно/self-hosted; работоспособность в изолированном контуре (AGSDS).
