# Эксплуатация и ввод в эксплуатацию — индекс (Stage 10)

Документы валидации, сертификации, развёртывания и промышленной эксплуатации ЕАСУР. Реализуют
подэтапы 10.1–10.10, ADR-024, RFC-0007 и стандарты Stage 10. Новая бизнес-функциональность на этапе
не создаётся (запрет этапа) — только верификация, испытания, сертификация, подготовка к эксплуатации
и ввод в действие.
