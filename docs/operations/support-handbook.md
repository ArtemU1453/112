# Support Handbook — ЕАСУР

- **Status:** APPROVED · Stage 10 · Реализует подэтап 10.8, Production Support Standard, Incident
  Escalation Standard.

## 1. Назначение
Руководство службы поддержки (L1–L3) промышленной эксплуатации: приём, диагностика, решение и
эскалация обращений и эксплуатационных инцидентов.

## 2. Уровни и обязанности
- **L1:** приём обращений, идентификация, известные решения, регистрация, первичная классификация.
- **L2:** углублённая диагностика по [Production Runbook](production-runbook.md), эксплуатационные действия.
- **L3:** инженерная поддержка (владельцы сервисов); устранение дефектов через релиз/RFC.

## 3. Классификация и SLA
Критичность обращения/инцидента (SEV-1..SEV-4, Incident Escalation Standard); целевые времена реакции
и восстановления; правила уведомления заявителей и заинтересованных сторон.

## 4. Обработка инцидентов
Регистрация → классификация → диагностика (runbooks/наблюдаемость) → устранение/обходное решение →
эскалация при необходимости → закрытие → пост-инцидентный разбор (SEV-1/2).

## 5. База знаний
Известные ошибки и решения, частые вопросы, ссылки на runbooks; актуализация по итогам инцидентов
(Long-Term Support Standard).

## 6. Границы
Исправления/оптимизация — через поддержку и релиз; новая функциональность — только полный цикл
(требование → RFC/ADR → релиз). Механизмы безопасности не отключаются.
