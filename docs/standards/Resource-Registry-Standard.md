# Resource Registry Standard — ЕАСУР

- **Status:** APPROVED · Stage 6 · Согласован с Geospatial Data Model, dispatch-service.

## Реестр сил и средств (6.3)
Объекты: пожарные части, аварийно-спасательные подразделения, техника, расчёты, экипажи, резервные
подразделения, специализированные группы, оборудование. Каждый — единый идентификатор (UUID).

## Атрибуты объекта
- **статус** (`UnitStatus` для техники/нарядов);
- **местоположение** (координаты/станция приписки);
- **готовность** (готов/на обслуживании/резерв);
- **специализация** (`UnitType`/`ServiceType`);
- **доступность** (текущая занятость);
- **ограничения** (эксплуатационные/технические).

## Правила
- Изменения реестра/местоположения — с фиксацией (аудит); история сохраняется.
- Реестр — источник данных для доступности (6.4) и рекомендаций (6.6).

## Definition of Done
Определены типы объектов, атрибуты (статус/местоположение/готовность/специализация/доступность/
ограничения) и единые идентификаторы.
