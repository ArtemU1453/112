# Knowledge Quality Standard — ЕАСУР

- **Status:** APPROVED · Stage 13 · Согласован с ADR-028, Knowledge Lifecycle Standard, Knowledge
  Analytics Guide, Data Quality Standard (Stage 8), Documentation Standards.

## Назначение
Требования к качеству единиц знаний (13.10): полнота, актуальность, точность, связность,
непротиворечивость. Обеспечивает доверие к базе знаний.

## Измерения качества
| Измерение | Критерий |
|-----------|----------|
| Полнота | Наличие обязательных атрибутов и разделов; покрытие тем |
| Актуальность | Соответствие действующим нормативам; срок пересмотра не истёк |
| Точность | Достоверность содержания; согласие с основаниями |
| Связность | Корректные ссылки/связи; отсутствие «битых» ссылок |
| Непротиворечивость | Отсутствие конфликтов между документами/правилами/SOP |
| Обоснованность | Правила имеют нормативное основание (Rule Authoring Standard) |

## Процессы контроля
- Автоматические проверки (ссылки/атрибуты/статусы) и экспертный обзор (владельцы знаний).
- Выявление устаревающих документов и пробелов — аналитикой (Knowledge Analytics); формирование
  рекомендаций по актуализации.
- Периодический пересмотр по срокам (Knowledge Lifecycle Standard).

## Реагирование
Дефекты качества регистрируются (при необходимости — техдолг/задача актуализации), устраняются через
утверждённый процесс публикации; критические (неактуальный норматив в применении) — приоритетно.

## Definition of Done
Определены измерения, процессы контроля и реагирование на дефекты качества знаний.
