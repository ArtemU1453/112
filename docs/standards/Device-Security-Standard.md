# Device Security Standard — ЕАСУР

- **Status:** APPROVED · Stage 7 · ADR-018 · Согласован с Mobile Security Standard.

## Управление устройствами (7.10, MDM)
- **Регистрация устройств** (enrollment) в MDM; привязка к учётной записи/подразделению.
- **Контроль соответствия политике безопасности** (compliance): версия ОС, шифрование, блокировка,
  отсутствие root/модификации.
- **Удалённая блокировка** и **удалённое стирание корпоративных данных** при утере/несоответствии.
- **Управление сертификатами** устройств/пользователей (согласовано с секретами, ADR-011).

## Правила
- MDM — через абстракцию (`MdmProvider`, ADR-018); без привязки к поставщику.
- Несоответствующие устройства ограничиваются/блокируются; события — в журнал безопасности.
- Распространение приложений — через MDM/внутренний канал (не обязательно публичные сторы).

## Definition of Done
Определены регистрация/compliance/блокировка/стирание/сертификаты через абстракцию MDM.
