# Decision Intelligence Standard — ЕАСУР

- **Status:** APPROVED · Stage 8 · ADR-021 · Согласован с AI Explainability / Statistical
  Methodology Standards, AI Assistance Policy (Stage 5).

## Возможности (8.4)
Выявление тенденций; обнаружение аномалий; прогноз нагрузки; рекомендации по распределению ресурсов;
выявление перегруженных подразделений; анализ зон риска.

## Правила (обязательные)
- **Каждый вывод сопровождается объяснением** и указанием использованных данных (AI Explainability
  Standard).
- **ИИ — только инструмент**; окончательное управленческое решение принимает человек (запрет
  автоматических решений на основе ИИ).
- Методики анализа документированы (Statistical Methodology Standard); модели — через адаптеры
  (ADR-021), без «чёрных ящиков» без методики.
- Результаты содержат показатель неопределённости/уверенности, где применимо.
- Рекомендации/выводы журналируются с входными данными и версией методики (аудит).

## Definition of Done
Определены тенденции/аномалии/прогноз/рекомендации/перегрузка/зоны риска с объяснением и участием человека.
