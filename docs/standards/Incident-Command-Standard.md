# Incident Command Standard — ЕАСУР

- **Status:** APPROVED · Stage 7 · Согласован с Emergency Operations (Stage 5), Operational
  Geography Standard (Stage 6).

## Приложения командования (7.2/7.3)
- **Начальник караула (7.2):** состав подразделения; техника; оборудование; задачи; журнал команд/
  событий; контроль выполнения; запрос дополнительных сил; изменение статуса работ.
- **Руководитель ликвидации ЧС (7.3):** управление силами; распределение задач; журнал решений;
  временная шкала; карта; контроль безопасности; журнал приказов; журнал рисков; контроль зон работ.

## Правила
- Все команды/приказы/решения — с фиксацией (журналы, синхронизируемые с платформенным аудитом).
- Запрос усиления/изменение состава — через общий контракт координации (Stage 6), по подтверждению
  ответственного (не автоматически).
- Контроль зон работ и безопасности — на карте (оперативная карта, Stage 6); риски журналируются.
- Приложения используют общий SDK/дизайн; бизнес-логика не дублируется.

## Definition of Done
Реализованы управление силами/задачи/журналы/временная шкала/карта/безопасность/приказы/риски/зоны.
