# Routing Standard — ЕАСУР

- **Status:** APPROVED · Stage 6 · ADR-014 · Согласован с Routing Governance Standard.

## Возможности (6.5)
- Построение маршрута; альтернативные маршруты; оценка времени прибытия (ETA); расчёт расстояния;
  учёт ограничений движения; временных перекрытий; сценариев объезда.

## Единый сервис
Все расчёты маршрутов — **через единый сервис маршрутизации** (`RoutingProvider`, ADR-014;
обязательная проверка этапа). Движок self-hosted, поставщик не фиксируется (TAP, `CAP-ROUTING-001`).

## Правила
- ETA/расстояние используются рекомендациями (6.6), но не приводят к автонаправлению нарядов.
- Ограничения/перекрытия учитываются актуальными данными; объезды предлагаются как альтернативы.
- Результаты маршрутизации журналируются и отображаются на оперативной карте.

## Definition of Done
Определены маршруты/альтернативы/ETA/расстояние/ограничения/перекрытия/объезды через единый сервис.
