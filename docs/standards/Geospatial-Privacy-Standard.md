# Geospatial Privacy Standard — ЕАСУР

- **Status:** APPROVED · Stage 6 · Согласован с Spatial Security Standard, Security Standards.

## Правила
- Минимизация персональных гео-данных (координаты заявителей, точные адреса) — хранить только
  необходимое; доступ по правам и с аудитом.
- Обезличивание/агрегация для аналитики (6.10) — точные персональные позиции не раскрываются в
  отчётах (агрегаты/тепловые карты по площадям).
- Сроки хранения точных гео-данных — по правовым требованиям (EGS/DGS); треки техники — служебные
  данные (не ПДн заявителя), но также под контролем доступа.
- Передача гео-ПДн вне контура запрещена (суверенность, AGSDS).

## Definition of Done
Определены минимизация/доступ/обезличивание/сроки хранения/запрет внешней передачи гео-ПДн.
