# Operational Forms Standard — ЕАСУР

- **Status:** APPROVED · Stage 7 · Согласован с Mobile Architecture, Validation Framework (Stage 4).

## Цифровые формы (7.7)
Рапорт; доклад; чек-листы; осмотр объекта; результаты разведки; информация о пострадавших;
использование оборудования; расход материалов.

## Правила
- Формы заполняются offline; сохраняются локально и синхронизируются (outbox, ADR-017).
- Валидация полей — Validation Framework (единые правила, полный перечень ошибок).
- Версионирование шаблонов форм; изменения шаблонов — управляемо (не ломают заполненные формы).
- Данные форм привязываются к инциденту/подразделению; действия журналируются.
- Формы используют общий контракт (`operational-forms-api.yaml`), не app-специфичный.

## Definition of Done
Определены типы форм, offline-заполнение, валидация, версионирование шаблонов, привязка и синхронизация.
