# Integration Certification Standard — ЕАСУР

- **Status:** APPROVED · Stage 9 · Согласован с Partner Integration Guide, Data Exchange Security
  Standard.

## Сертификация интеграции (9.10)
Перед вводом интеграции в продуктив подтверждается соответствие:
- контракту (Contract Testing Standard) и канонической модели;
- требованиям безопасности (mTLS/подпись/scope, ADR-023);
- SLA (Integration SLA Standard);
- наблюдаемости (трассировка/мониторинг/аудит);
- регистрации в реестре внешних систем.

## Процесс
Заявка → тесты в песочнице → проверка безопасности → проверка SLA/наблюдаемости → решение о
сертификации → продуктив. Пересертификация — при несовместимых изменениях контракта/безопасности.

## Definition of Done
Определены критерии и процесс сертификации интеграций перед продуктивом и при изменениях.
