# BI Security Standard — ЕАСУР

- **Status:** APPROVED · Stage 8 · Согласован с Security Architecture, Geospatial Privacy Standard.

## Правила
- Доступ к дашбордам/отчётам/данным — по RBAC и **уровню управления** (row/area-level security):
  пользователь видит только разрешённую область (регион/подразделение/уровень).
- Чувствительные данные (ПДн, точные координаты, персонал) — минимизация/обезличивание в аналитике
  (Geospatial/Data Privacy стандарты).
- Данные защищены при хранении (DWH) и передаче (TLS); экспорт — с контролем прав и аудитом.
- Доступ к аналитике журналируется (Mobile/Operational Audit); аномалии доступа — алерты.
- Суверенно/self-hosted (AGSDS); передача аналитики вовне контролируется.

## Definition of Done
Определены RBAC/уровневый доступ, минимизация чувствительных данных, защита хранения/передачи, аудит.
