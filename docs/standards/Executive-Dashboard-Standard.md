# Executive Dashboard Standard — ЕАСУР

- **Status:** APPROVED · Stage 8 · ADR-020 · Согласован с Dashboard Design / BI Security Standards.

## Дашборды по уровням (8.2)
Диспетчер; старший диспетчер; оперативный дежурный; руководитель гарнизона; областное управление;
центральный аппарат. Каждый уровень — релевантный набор показателей и область видимости.

## Возможности
Показатели в реальном времени; исторические данные; сравнение периодов; drill-down; фильтрация;
экспорт.

## Правила
- Доступ и область данных — по RBAC и уровню управления (BI Security Standard); минимизация
  чувствительных данных.
- Показатели — из каталога метрик (единые определения); происхождение прослеживается.
- Единый визуальный стиль (Dashboard Design Standard); согласованные единицы/периоды.
- Экспорт — с сохранением контекста (период/фильтры) для воспроизводимости.

## Definition of Done
Определены дашборды для всех уровней управления с real-time/history/сравнением/drill-down/фильтром/
экспортом и разграничением доступа.
