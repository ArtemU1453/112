# Field Operations Standard — ЕАСУР

- **Status:** APPROVED · Stage 7 · Согласован с Emergency Operations (Stage 5), Mobile Architecture.

## Полевое приложение подразделения (7.1)
Возможности: получение карточки выезда; маршрут и навигация (единый сервис, ADR-014); информация об
объекте; статус подразделения (`UnitStatus`); журнал действий; отправка статусов; прикрепление
фото/видео/документов/координат (Media Governance Standard).

## Правила
- Отправка статусов и медиа — offline-first (outbox, ADR-017); журналируется.
- Смена статуса подразделения обновляет платформу (события `unit.status-changed`) при синхронизации.
- Навигация/маршруты — через общий сервис маршрутизации (не app-специфичный).
- Действия в поле фиксируются в мобильном аудите и синхронизируются с платформенным аудитом.

## Безопасность
Данные шифруются при хранении и передаче; секреты не в коде; сессия завершается по политике.

## Definition of Done
Реализованы карточка/маршрут/навигация/объект/статусы/журнал/медиа/координаты в offline-first.
