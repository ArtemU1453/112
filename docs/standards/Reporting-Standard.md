# Reporting Standard — ЕАСУР

- **Status:** APPROVED · Stage 8 · Согласован с Analytics Architecture, Document Lifecycle
  Management Standard.

## Отчётность (8.5)
Шаблоны отчётов; регламентные отчёты; пользовательские отчёты; автоматическое расписание; **версии
отчётов**; цифровая подпись (если применяется архитектурой); экспорт (PDF/CSV/…).

## Правила (обязательные)
- **Воспроизводимость:** отчёт формируется из фиксированного среза DWH (версия/период); повторное
  построение даёт идентичный результат.
- **Происхождение:** каждый показатель отчёта прослеживается до источника (Data Lineage Standard).
- Версионирование отчётов и шаблонов; изменения шаблона не ломают ранее сформированные отчёты.
- Регламентные отчёты — по расписанию, с журналированием формирования и доступа.
- Экспорт/подпись — через платформенные механизмы (Document Platform, File Storage).

## Definition of Done
Определены шаблоны/регламентные/пользовательские/расписание/версии/подпись/экспорт и
воспроизводимость/происхождение.
