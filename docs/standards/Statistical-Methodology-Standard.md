# Statistical Methodology Standard — ЕАСУР

- **Status:** APPROVED · Stage 8 · Согласован с Analytics Governance / Data Lineage Standards.

## Правила (обязательные)
- **Любая аналитическая модель/метод используется только с документированной методикой** (запрет
  моделей без методики).
- Методика описывает: постановку задачи, данные и их подготовку, метод/формулы, допущения,
  ограничения, метрики качества, интерпретацию.
- Методики версионируются; изменение методики — новая версия с оценкой влияния.
- Воспроизводимость: фиксируются данные (срез), параметры и версия метода.
- Корректность: избегать смещений (bias), некорректных агрегаций, ложных корреляций; указывать
  ограничения.

## Definition of Done
Определены обязательность/состав/версионирование/воспроизводимость методик и требования корректности.
