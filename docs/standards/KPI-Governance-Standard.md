# KPI Governance Standard — ЕАСУР

- **Status:** APPROVED · Stage 8 · Согласован с Metrics Catalog / Data Lineage Standards.

## KPI (8.3)
Минимальный набор (конфигурируемый): время ответа; время обработки вызова; время выезда; время
прибытия; длительность ликвидации; нагрузка подразделений; эффективность реагирования; выполнение
нормативов.

## Правила (обязательные)
- **Автоматический расчёт** всех KPI из DWH (без ручного ввода).
- **Конфигурируемость:** формулы/пороги/нормативы задаются конфигурацией (не хардкод).
- Каждый KPI зарегистрирован в каталоге метрик (определение, формула, источник, владелец —
  Metrics Catalog Standard) и прослеживается (Data Lineage Standard).
- Единые определения KPI для всех уровней управления (сопоставимость).
- Изменение методики расчёта KPI — версионируется; исторические значения пересчитываются
  контролируемо (с аудитом).

## Definition of Done
Определён набор KPI, автоматический расчёт, конфигурируемость, регистрация в каталоге и lineage.
