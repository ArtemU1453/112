# Cross-System Audit Standard — ЕАСУР

- **Status:** APPROVED · Stage 9 · Согласован с Audit Platform (Stage 4), Integration Security
  Architecture. Immutable Audit.

## Правила
- Каждый межсистемный обмен фиксируется в **неизменяемом** аудите: источник, назначение, интерфейс,
  версия контракта, идентификатор/`correlationId` сообщения, результат, время, статус безопасности
  (mTLS/подпись).
- Аудит доступа внешних систем (аутентификация/авторизация/отказы) — обязателен; обмен без аудита
  запрещён.
- Записи append-only; изменение/удаление — запрещено (полная прослеживаемость).
- Аудит доступен для расследований и сертификации; коррелируется с трассировкой (OpenTelemetry).

## Definition of Done
Определён неизменяемый межсистемный аудит обменов и доступа с полной прослеживаемостью.
