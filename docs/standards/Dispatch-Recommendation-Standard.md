# Dispatch Recommendation Standard — ЕАСУР

- **Status:** APPROVED · Stage 6 · Согласован с Emergency Operations (Stage 5), AI Assistance Policy.

## Механизм рекомендаций (6.6)
Учитывает: расстояние; время прибытия (ETA, ADR-014); специализацию (`UnitType`); готовность;
резерв; тип происшествия (`IncidentType`); уровень сложности; нормативы реагирования.

## Правило (обязательное)
**Система только рекомендует. Окончательное решение принимает диспетчер.** Рекомендация имеет
статус PROPOSED; направление нарядов — только после APPROVED/MODIFIED диспетчером (запрет
автонаправления, Operational State Machine §5).

## Прозрачность
Для рекомендации отображаются факторы (расстояние/ETA/специализация/готовность) и норматив; выбор и
изменение журналируются (Operational Audit).

## Definition of Done
Определены факторы рекомендации и правило обязательного подтверждения диспетчером.
