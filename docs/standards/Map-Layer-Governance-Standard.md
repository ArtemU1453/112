# Map Layer Governance Standard — ЕАСУР

- **Status:** APPROVED · Stage 6 · Согласован с GIS Architecture, ADR-015.

## Управление слоями (6.1)
- Слои: базовые (адм. границы, населённые пункты, улицы, здания) и тематические (зоны
  ответственности, инциденты, наряды, маршруты, тепловые карты).
- Каждый слой имеет: идентификатор, тип геометрии, источник, CRS, права доступа, видимость.
- Источник тайлов/данных — self-hosted (`MapDataProvider`, ADR-015); привязка к внешнему
  поставщику запрещена (TAP).

## Правила
- Порядок отрисовки и видимость управляются декларативно; тематические слои переключаемы.
- Изменение состава/источника слоёв — через управляемый процесс; чувствительные слои — по правам
  (Spatial Security / Geospatial Privacy Standards).

## Definition of Done
Определены управление слоями, метаданные, источники (self-hosted) и права доступа.
