# Spatial Security Standard — ЕАСУР

- **Status:** APPROVED · Stage 6 · Согласован с Security Architecture, Geospatial Privacy Standard.

## Правила
- Доступ к пространственным данным/слоям — по RBAC/ABAC (роли/подразделение/зона); чувствительные
  слои (расположение сил, объекты) — ограниченный доступ.
- **Координаты объектов меняются только с фиксацией** изменений (версия/аудит; запрет незаметного
  изменения).
- **Историю перемещений удалять запрещено** (append-only).
- Гео-данные передаются по TLS; в air-gapped — без внешних сервисов (ADR-015, AGSDS).
- Все пространственные действия (просмотр чувствительных слоёв, изменение координат, трекинг)
  журналируются (Operational Audit).

## Definition of Done
Определены контроль доступа к гео-данным, фиксация изменений координат, неизменяемость истории,
защита передачи и аудит.
