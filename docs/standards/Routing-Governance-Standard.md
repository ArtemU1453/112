# Routing Governance Standard — ЕАСУР

- **Status:** APPROVED · Stage 6 · ADR-014 · Согласован с Routing Standard, TVMS.

## Управление
- Движок маршрутизации и дорожные данные — self-hosted; версии — каталог версий (TVMS).
- Смена движка/данных — обновлением адаптера и Technology Mapping (без изменения потребителей);
  несовместимое изменение контракта — RFC/ADR.
- Обновление дорожных данных — регламентно (в т. ч. офлайн для air-gapped, AGSDS).

## Качество и производительность
Целевые показатели времени расчёта маршрута — GIS Performance Standard; мониторинг —
Operational Telemetry Standard.

## Definition of Done
Определены управление движком/данными, версионирование, обновление и качество маршрутизации.
