# Offline Data Governance Standard — ЕАСУР

- **Status:** APPROVED · Stage 7 · Согласован с DGS, Offline Synchronization Standard, AGSDS.

## Управление офлайн-данными
- Что кэшируется локально: только необходимые для полевой работы данные (минимизация); чувствительные
  данные — с шифрованием и ограничением объёма/срока.
- Срок жизни локальных данных: очистка устаревших/неактуальных; удаление при выходе/стирании (MDM).
- Целостность: версии и контрольные суммы (ADR-017); отсутствие потери данных.
- Право доступа к офлайн-данным — по роли/устройству; журналирование доступа.

## Правила
- Реальные ПДн вне необходимости не кэшируются; при потере устройства — удалённое стирание
  корпоративных данных (MDM, ADR-018).
- Резервирование серверной стороны — Backup Policy; клиент — источник изменений через синхронизацию.

## Definition of Done
Определены состав/срок/защита/доступ офлайн-данных и их очистка/стирание.
