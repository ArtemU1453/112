# Data Retention for Analytics Standard — ЕАСУР

- **Status:** APPROVED · Stage 8 · Согласован с EGS, DGS, Backup Policy, Geospatial Privacy Standard.

## Правила хранения аналитических данных
- Сроки хранения по классам данных: сырые (staging), интегрированные (core), витрины, отчёты —
  задаются политикой (правовые/ведомственные требования).
- Исторические данные **неизменяемы**; изменения — только с полным аудитом (запрет этапа);
  архивирование по истечении оперативного срока.
- Обезличивание/агрегация для долгосрочного хранения чувствительных данных (Privacy стандарты).
- Резервное копирование аналитического хранилища — Backup Policy; суверенно/self-hosted (AGSDS).
- Удаление/архивирование — контролируемо и журналируемо.

## Definition of Done
Определены сроки по классам, неизменяемость истории, обезличивание, резервирование и аудит.
