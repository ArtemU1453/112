# Canonical Identifier Standard — ЕАСУР

- **Status:** APPROVED · Stage 9 · Согласован с Canonical Data Model, Master Data Governance Standard.

## Правила
- Все канонические сущности имеют **стабильные, уникальные** идентификаторы (UUID/URN-схема).
- Сопоставление внешних идентификаторов с каноническими ведётся в платформе (mapping), с сохранением
  истории (прослеживаемость).
- Идентификаторы не переиспользуются и не изменяются; изменение — только через управляемый процесс с
  аудитом.
- Идентификаторы справочников/НСИ — из единого источника (Reference Data Management Standard).

## Definition of Done
Определены схема канонических идентификаторов, сопоставление внешних, стабильность и прослеживаемость.
