# Geospatial Data Governance Standard — ЕАСУР

- **Status:** APPROVED · Stage 6 · Согласован с DGS, Canonical Data Model Standard, ADR-003/015.

## Управление гео-данными
- Источники данных (адм. границы, населённые пункты, улицы, здания, дороги, тайлы) — self-hosted,
  открытые/ведомственные; привязка к поставщику запрещена (TAP).
- Единые идентификаторы (UUID) и CRS обмена (CRS Standard); качество/полнота контролируются.
- Обновление данных — регламентно, версионируемо (в т. ч. офлайн, AGSDS); журнал обновлений.
- История изменений гео-объектов и перемещений — append-only (запрет удаления).
- Резервное копирование гео-данных — Backup Policy; целостность — контрольные суммы.

## Управление изменениями
Изменения модели/справочников/источников — через RFC/ADR при влиянии на контракты/схему;
синхронизация Technology Mapping и документации.

## Definition of Done
Определены источники/идентификаторы/CRS/качество/обновление/история/резервирование гео-данных.
