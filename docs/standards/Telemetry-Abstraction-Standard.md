# Telemetry Abstraction Standard — ЕАСУР

- **Status:** APPROVED · Stage 6 · ADR-015 · Согласован с Telemetry Integration Standard, TAP.

## Абстракция телеметрии
Единый порт `TelemetrySource` для приёма позиций/событий движения от разнородных источников.
Нормализация в каноническую модель (позиция WGS84, время UTC, скорость/курс, источник).

## Правила
- Прикладной код не зависит от конкретного протокола/поставщика телеметрии (запрет vendor lock).
- Адаптеры инкапсулируют форматы/протоколы; добавление источника — новый адаптер + Technology
  Mapping.
- Надёжность приёма (буферизация/повтор) — согласно Messaging/Integration стандартам.

## Definition of Done
Определён порт телеметрии, каноническая нормализация и правила подключения источников адаптерами.
