# Media Governance Standard — ЕАСУР

- **Status:** APPROVED · Stage 7 · Согласован с File Storage Abstraction (ADR-013), Document
  Lifecycle Management Standard.

## Медиа (7.8)
Работа с фото, видео, голосовыми заметками, документами. Для каждого объекта: **контроль
целостности** (контрольная сумма), **метаданные** (время/координаты/автор/устройство), **привязка
к инциденту**.

## Правила
- Медиа сохраняется offline (шифрованно) и синхронизируется в объектное хранилище (ADR-013) при
  связи; передача — по TLS.
- Целостность проверяется при загрузке и хранении; изменение/удаление — по политикам (запрет
  удаления в обход политик хранения).
- Метаданные не содержат лишних ПДн; доступ — по правам (Security/Privacy стандарты).
- Крупные медиа — приоритетная/фоновая синхронизация (Battery/Performance стандарты).

## Definition of Done
Определены типы медиа, целостность, метаданные, привязка к инциденту, offline-хранение и синхронизация.
