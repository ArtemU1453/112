# Message Reliability Standard — ЕАСУР

- **Status:** APPROVED · Stage 9 · Согласован с Messaging Architecture (Stage 5), ADR-008.

## Надёжность обмена (9.4)
- **Гарантированная доставка:** at-least-once с подтверждением обработки; персистентность.
- **Повторная доставка:** политика повторов (backoff) при сбое; DLQ при исчерпании.
- **Дедупликация:** ключи идемпотентности; повторная доставка не приводит к дублям обработки.
- **Контроль целостности:** контрольные суммы/подпись (ADR-023).
- **Подтверждение обработки:** ack/nack; отправитель уведомляется о результате.
- Синхронный и асинхронный обмен — по контракту; асинхронный предпочтителен для слабой связанности.

## Правила
Потери сообщений отслеживаются (Integration Monitoring Guide) и расследуются; отсутствие потерь —
цель. Идемпотентность обязательна для повторяемых операций.

## Definition of Done
Определены гарантированная доставка/повторы/дедупликация/целостность/подтверждение обработки.
