# Regulatory Document Standard — ЕАСУР

- **Status:** APPROVED · Stage 13 · Согласован с ADR-028, Document Versioning Standard, Knowledge
  Publication Standard, Document Lifecycle Management Standard (Stage 4), EAGF.

## Назначение
Правила управления нормативными документами (13.2): версии, статусы, утверждение, публикация,
архивирование, срок действия, журнал ознакомления. Используются только утверждённые документы.

## Жизненный цикл документа
```
Draft → Under Review → Approved → Published → (Superseded | Expired) → Archived
```
- **Draft/Under Review** — подготовка и согласование; не применяется в работе.
- **Approved → Published** — утверждён уполномоченным лицом и опубликован (Knowledge Publication Standard).
- **Superseded** — заменён новой версией; **Expired** — истёк срок действия; **Archived** — в архиве.
- История статусов и версий **неизменяема** (удаление истории запрещено).

## Обязательные атрибуты
Идентификатор, название, тип, версия (Document Versioning Standard), статус, автор/владелец, орган
утверждения, даты (создания/утверждения/публикации/окончания действия), основание, связи.

## Ознакомление
Для документов обязательного ознакомления ведётся **журнал ознакомления** (кто/когда ознакомился,
подтверждение); контроль прохождения — Knowledge Distribution (13.8).

## Правила
- Применение неутверждённых документов запрещено.
- Публикация/изменение — только через утверждённый процесс согласования.
- Срок действия контролируется; истёкшие документы не применяются (перевод в Expired).

## Definition of Done
Определены жизненный цикл, атрибуты, версионирование, ознакомление и правила нормативных документов.
