# Field Telemetry Standard — ЕАСУР

- **Status:** APPROVED · Stage 7 · ADR-015/018 · Согласован с Telemetry Abstraction Standard (Stage 6).

## Полевая телеметрия (7.9)
Источники через **адаптеры**: GPS; терминалы; бортовые системы; датчики; носимые устройства (если
используются); внешние источники координат. Единый порт (`TelemetrySource`, ADR-015).

## Правила
- Прикладная логика отделена от телеметрии (адаптеры); нормализация в каноническую модель
  (позиция WGS84, время UTC, источник).
- Данные телеметрии — offline-буферизация и синхронизация; история перемещений append-only
  (запрет удаления).
- Разные источники подключаются новыми адаптерами (без vendor lock); частота/точность — под
  Battery/Performance стандарты.

## Definition of Done
Определены источники через адаптеры, нормализация, буферизация/синхронизация, неизменяемость истории.
