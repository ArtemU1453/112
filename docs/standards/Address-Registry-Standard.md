# Address Registry Standard — ЕАСУР

- **Status:** APPROVED · Stage 6 · Согласован с Address Intelligence Standard, Geospatial Data Model.

## Адресный реестр (6.1)
- Единый реестр адресов: регион → район → населённый пункт → улица → дом (+корпус) → координаты.
- Каждый адрес — единый идентификатор (UUID); связь со зданием/улицей/населённым пунктом.
- Источник данных — self-hosted (открытые/ведомственные данные); привязка к поставщику запрещена.

## Правила
- **История изменений** адресов ведётся (append-only); удаление исторических записей запрещено.
- Изменение адреса/координат — только с фиксацией (версия/аудит).
- Качество данных — контроль полноты/дубликатов (GIS Performance / Data Governance Standards).

## Definition of Done
Определены структура реестра, идентификаторы, история и правила изменения.
