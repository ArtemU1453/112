# Production Support Standard — ЕАСУР

- **Status:** APPROVED · Stage 10 · Согласован с Platform Operations Framework, Support Handbook,
  Incident Escalation Standard, Long-Term Support Standard.

## Назначение
Организация поддержки промышленной эксплуатации (10.8). Определяет уровни, роли, SLA и процессы
обработки обращений и инцидентов эксплуатации.

## Уровни поддержки
- **L1** — приём обращений, первичная диагностика, известные решения (Support Handbook).
- **L2** — углублённая диагностика, эксплуатационные процедуры (Production Runbook).
- **L3** — инженерная поддержка (владельцы сервисов), устранение дефектов, изменения через RFC/релиз.

## Процессы
- Классификация обращений/инцидентов по критичности; целевые времена реакции/восстановления (SLA).
- Эскалация — Incident Escalation Standard; крупные сбои — DR/Business Continuity.
- Управление проблемами (root cause), знаниями (база решений), запросами на изменение (через релиз).
- Наблюдаемость: дежурные дашборды/алерты; отчётность по SLA/SLO.

## Ограничения
Исправления дефектов и оптимизация — допускаются; новая бизнес-функциональность — только через
полный цикл (требование → RFC/ADR → релиз). Механизмы безопасности не отключаются.

## Definition of Done
Определены уровни, процессы, SLA и границы поддержки промышленной эксплуатации.
