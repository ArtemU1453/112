# Contract Testing Standard — ЕАСУР

- **Status:** APPROVED · Stage 9 · Согласован с Testing Strategy (Stage 4), Partner Onboarding Standard.

## Контрактное тестирование (9.9)
- Проверка соответствия реализации контракту (provider/consumer contract testing).
- Тесты совместимости версий (обратная/прямая совместимость).
- Автоматический прогон при изменении контракта/реализации (CI); публикация — только при зелёных
  контрактных тестах.
- Партнёрская песочница (Partner Sandbox) — эмуляция сервисов и тестовые данные для проверки
  интеграций перед продуктивом.

## Правила
Контракт — источник истины; несовместимость выявляется тестами до публикации; провайдер и
потребители синхронизированы по версии.

## Definition of Done
Определены provider/consumer contract testing, тесты совместимости, автоматизация и песочница.
