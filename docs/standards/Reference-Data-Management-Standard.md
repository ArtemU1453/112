# Reference Data Management Standard — ЕАСУР

- **Status:** APPROVED · Stage 9 · Согласован с Master Data Governance Standard (Stage 8),
  Canonical Data Model.

## НСИ (9.8)
Нормативно-справочная информация (справочники): версии справочников; публикация изменений;
синхронизация; аудит изменений; проверка согласованности.

## Правила
- Единый источник истины для справочников; версионирование (каждая версия — неизменяема, публикуется).
- Публикация изменений — контролируемо (кто/когда/что), с уведомлением потребителей.
- Синхронизация справочников с внешними системами — через интеграционную платформу (канонические
  идентификаторы).
- Аудит изменений — неизменяемый; проверка согласованности (непротиворечивость/полнота).
- Обратная совместимость: старые версии доступны на переходный период.

## Definition of Done
Определены версии/публикация/синхронизация/аудит/согласованность НСИ.
