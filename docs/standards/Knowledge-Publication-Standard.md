# Knowledge Publication Standard — ЕАСУР

- **Status:** APPROVED · Stage 13 · Согласован с ADR-028, Regulatory Document Standard, Knowledge
  Distribution (13.8), Release Governance Standard (аналогия gate'ов).

## Назначение
Процесс согласования и публикации единиц знаний (13.2, 13.8): кто и как утверждает и публикует
документы/SOP/правила/справочники, как распространяются изменения.

## Процесс публикации
```
Подготовка → Согласование → Утверждение → Публикация → Распространение → Контроль ознакомления
```
- Публикация — только после утверждения уполномоченным органом; неутверждённое не публикуется и не
  применяется.
- Публикация без утверждённого процесса согласования запрещена (запрет этапа).

## Распространение изменений (Knowledge Distribution)
- Уведомление затронутых пользователей; для обязательных документов — **обязательное ознакомление**.
- **Журнал ознакомления**: кто/когда ознакомился, подтверждение получения; контроль прохождения.
- Крупные (MAJOR) изменения инициируют переознакомление и, при необходимости, переаттестацию
  (Competency Management Standard).

## Трассируемость
Каждая публикация фиксирует версию, утвердившего, дату, основание, охват распространения — неизменяемо.
История публикаций не удаляется.

## Definition of Done
Определены процесс согласования/публикации, распространение, обязательное ознакомление и трассируемость.
