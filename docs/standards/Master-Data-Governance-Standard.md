# Master Data Governance Standard — ЕАСУР

- **Status:** APPROVED · Stage 8 · Согласован с Canonical Data Model Standard (Stage 4), DGS.

## Мастер-данные
Справочники и эталонные сущности (подразделения, типы происшествий, регионы, нормативы, персонал/
роли) — единые, версионируемые, с владельцем.

## Правила
- Единый источник истины для мастер-данных; аналитика использует эталонные справочники (сопоставимость).
- Разрешение дубликатов и сопоставление сущностей — по правилам мастер-данных (Data Quality Standard).
- Изменения мастер-данных — версионируются и аудируются; влияние на историю оценивается.
- Согласованность с канонической моделью (Stage 4) и оперативными справочниками (User Management).

## Definition of Done
Определены состав мастер-данных, единый источник, версионирование, разрешение дубликатов.
