# Analytics Governance Standard — ЕАСУР

- **Status:** APPROVED · Stage 8 · Согласован с EAGF, Data Governance стандартами.

## Управление аналитикой
- Единое аналитическое пространство (`analytics-service` + DWH); **дублирующие аналитические
  платформы запрещены** (запрет этапа).
- Каждый аналитический продукт (KPI/отчёт/дашборд/модель) имеет владельца, определение, источник,
  методику и версию.
- Изменения методик/определений — версионируются; влияние на исторические значения оценивается
  (пересчёт с аудитом).
- Доступ к аналитике — по RBAC/уровню (BI Security Standard); данные защищены.

## Правила
- Модели/методы — с документированной методикой (Statistical Methodology Standard); без методики
  использовать запрещено.
- Происхождение данных не скрывается (Data Lineage Standard).
- ИИ — вспомогательный инструмент; автоматические управленческие решения на основе ИИ запрещены.

## Definition of Done
Определены единость аналитики, владение/версионирование продуктов, методики, доступ и запреты.
