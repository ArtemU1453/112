# Dashboard Design Standard — ЕАСУР

- **Status:** APPROVED · Stage 8 · Согласован с Executive Dashboard Standard, UX.

## Правила визуализации
- Единый визуальный язык: согласованные цвета состояний (приоритеты/статусы — как в дизайн-системе),
  единицы измерения, форматы времени (UTC/локально), периоды.
- Иерархия информации: ключевые показатели — заметно; детализация — через drill-down.
- Понятность: подписи, легенды, единицы; отсутствие вводящих в заблуждение визуализаций (корректные
  шкалы/агрегации).
- Доступность: контраст, читаемые шрифты, дневной/ночной режимы (согласовано с Accessibility).
- Экспорт сохраняет контекст (период/фильтры) для воспроизводимости.

## Definition of Done
Определены единый визуальный язык, иерархия, понятность, доступность и экспорт дашбордов.
