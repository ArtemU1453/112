# Forecasting Standard — ЕАСУР

- **Status:** APPROVED · Stage 8 · Согласован с Forecast Validation / Statistical Methodology
  Standards, ADR-021.

## Прогнозирование (8.7)
Прогноз количества вызовов; прогноз нагрузки; прогноз потребности в силах; прогноз сезонных
изменений; сценарный анализ.

## Правила (обязательные)
- **Все прогнозы содержат показатели неопределённости** (доверительные интервалы/ошибку).
- Прогноз **не является единственным основанием** для принятия решений (вспомогательный характер).
- Методика прогноза документирована (Statistical Methodology Standard); валидация точности —
  Forecast Validation Standard (backtesting, метрики ошибки).
- Модели — через адаптеры (ADR-021); входные данные и версия модели фиксируются (воспроизводимость,
  lineage).
- Сценарный анализ — с явными допущениями.

## Definition of Done
Определены прогнозы вызовов/нагрузки/сил/сезонности/сценариев с неопределённостью и валидацией.
