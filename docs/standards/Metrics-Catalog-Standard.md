# Metrics Catalog Standard — ЕАСУР

- **Status:** APPROVED · Stage 8 · Согласован с KPI Governance / Data Lineage Standards.

## Каталог метрик
Единый реестр показателей: имя, определение, формула, единицы, гранулярность, источник, владелец,
версия, статус. Все KPI/показатели дашбордов регистрируются в каталоге.

## Правила
- Единые определения метрик для всех уровней управления (сопоставимость).
- Каждая метрика прослеживается до источника (Data Lineage Standard) и имеет методику.
- Изменение определения метрики — новая версия; влияние на отчёты/дашборды оценивается.
- Дублирование метрик с разными определениями запрещено (единый источник истины по показателям).

## Definition of Done
Определён реестр метрик с определениями/формулами/источниками/владельцами/версиями.
