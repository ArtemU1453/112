# Exercise Data Governance Standard — ЕАСУР

- **Status:** APPROVED · Stage 12 · Согласован с ADR-026, Synthetic Data Standard, Data Retention for
  Analytics Standard (Stage 8), Simulation Security Standard, Exercise Management Guide.

## Назначение
Управление данными учений и моделирования (12.8, 12.10): происхождение, маркировка, хранение,
ретенция, разделение с реальными данными.

## Принципы
- **Разделение сред:** данные учений/моделирования хранятся отдельно от промышленных; смешение
  учебных и реальных инцидентов запрещено.
- **Маркировка:** все записи помечены как синтетические/учебные (origin=simulation); синтетические
  данные не выдаются за реальные.
- **Происхождение и версии:** привязка к версии сценария/двойника и seed (воспроизводимость).
- **Ретенция:** история экспериментов и результатов сохраняется (Simulation Analytics) по политике
  ретенции; удаление — по регламенту, не спонтанно.
- **Приватность:** обезличивание при формировании наборов (Synthetic Data Standard).

## Хранение и доступ
Хранилище истории экспериментов с версионированием; доступ по ролям (изолированный контур); аудит
доступа. Экспорт отчётов — контролируемый, с маркировкой источника.

## Definition of Done
Определены разделение сред, маркировка, происхождение, ретенция и приватность данных учений.
