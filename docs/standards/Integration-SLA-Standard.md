# Integration SLA Standard — ЕАСУР

- **Status:** APPROVED · Stage 9 · Согласован с Integration Monitoring Guide, External Systems
  Registry Specification.

## Правила
- Для каждой интеграции определяется SLA: доступность канала, целевая латентность (p95/p99),
  пропускная способность, окно поддержки, сроки уведомления об изменениях/депрекации.
- SLA фиксируется в записи реестра внешних систем; критичность влияет на целевые показатели/алерты.
- Соблюдение SLA контролируется мониторингом (Integration Monitoring Guide); нарушения — алерты и
  инциденты (Platform Operations Framework).
- Депрекация версий API — с соблюдением сроков SLA уведомления.

## Definition of Done
Определены состав SLA, привязка к реестру, контроль соблюдения и обработка нарушений.
