# Telemetry Integration Standard — ЕАСУР

- **Status:** APPROVED · Stage 6 · ADR-015 · Согласован с Telemetry Abstraction Standard.

## Отслеживание ресурсов (6.8)
События: местоположение техники; движение; прибытие; убытие; завершение задания; возврат в
подразделение. Источник — телеметрия через **адаптеры** (`TelemetrySource`, ADR-015); прикладная
логика отделена от телеметрии (обязательная проверка этапа).

## Правила
- Разные источники (GPS/АРМ экипажа/внешние трекеры) — через единый порт; замена без изменения
  потребителей (TAP, `CAP-TELEMETRY-001`).
- **Историю перемещений удалять запрещено** (append-only); треки неизменяемы.
- События трекинга обновляют статусы нарядов и оперативную карту; журналируются.

## Definition of Done
Определены события трекинга, адаптеры источников, отделение от логики, неизменяемость истории.
