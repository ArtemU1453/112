# Forecast Validation Standard — ЕАСУР

- **Status:** APPROVED · Stage 8 · Согласован с Forecasting / Statistical Methodology Standards.

## Валидация прогнозов
- Оценка точности на исторических данных (backtesting); метрики ошибки (например, MAE/MAPE/RMSE).
- Доверительные интервалы/показатели неопределённости обязательны для каждого прогноза.
- Регулярная переоценка качества моделей; деградация — триггер пересмотра методики.
- Сравнение с базовыми (наивными) моделями для обоснования сложности.

## Правила
- Прогноз без оценки достоверности не публикуется (обязательная проверка этапа).
- Методика валидации документирована и версионируется; результаты валидации фиксируются.
- Прогноз — вспомогательный; не единственное основание решений.

## Definition of Done
Определены метрики точности, интервалы неопределённости, регулярная переоценка и базовые сравнения.
