# Battery Optimization Standard — ЕАСУР

- **Status:** APPROVED · Stage 7 · Согласован с Field Telemetry / Mobile Performance Standards.

## Требования
- Управление частотой GPS/телеметрии по контексту (в движении/на месте/в резерве) — баланс точности
  и энергопотребления.
- Синхронизация — батчами по расписанию/событиям, а не постоянные соединения; уважение системных
  ограничений фоновой работы.
- Экономичный режим при низком заряде: снижение частоты телеметрии/синхронизации, приоритет
  критичных функций.
- Минимизация wake-lock; эффективное использование сети (дельта/сжатие).

## Правила
Функции жизнеобеспечения (приём приказов/тревог) не отключаются; экономия не в ущерб безопасности
реагирования.

## Definition of Done
Определены адаптивная телеметрия/синхронизация, экономичный режим и приоритет критичных функций.
