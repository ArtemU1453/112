# Configuration Standards — ЕАСУР

- **Status:** APPROVED · Baseline: v1.0

Единые правила конфигурации (согласовано с Environment Governance Standard).

## Источники конфигурации и приоритет
Приоритет (от высшего): переменные окружения → внешняя конфигурация профиля → значения по
умолчанию в артефакте. Значения по умолчанию безопасны для локальной разработки, но не для prod.

## ENV (переменные окружения)
- Именование `UPPER_SNAKE_CASE` (`DB_URL`, `KAFKA_BOOTSTRAP_SERVERS`, `KEYCLOAK_ISSUER_URI`).
- Все внешние адреса/параметры инжектируются через ENV; в коде — только имена переменных.
- Обязательные переменные документируются в `environments/` и проверяются перед развёртыванием (EGS).

## Properties / YAML
- Приложения используют `application.yml` со ссылками `${ENV:default}`.
- Отступ YAML — 2 пробела; без табов; без секретов (только плейсхолдеры).
- Единая структура ключей между сервисами (datasource/kafka/redis/security/management).

## Secrets
- Секреты **не** хранятся в Git/Dockerfile/образах/открытых конфигах (RGS, EGS).
- Источник — внешний секрет-менеджер (`CAP-SECRETS-001`) или Kubernetes Secret/External Secrets.
- В репозитории — только шаблоны с плейсхолдерами и описанием требуемых ключей.

## Profiles
- Профили окружений (`local`, `dev`, `test`, `stage`, `prod`, `k8s`) активируются через
  `SPRING_PROFILES_ACTIVE`/эквивалент.
- Профиль изменяет только окружение-специфичные параметры; бизнес-поведение неизменно.

## Feature Flags
- Управляемое включение/отключение функциональности без пересборки.
- Флаги — часть конфигурации окружения (EGS), версионируются, имеют владельца и срок жизни.
- Временные флаги удаляются после стабилизации функциональности (контроль техдолга).

## Валидация конфигурации
- Приложение при старте валидирует наличие обязательных параметров и завершает работу при их
  отсутствии (fail-fast).
- Согласованность конфигураций окружений проверяется автоматически (EGS «Проверка соответствия»).

## Версии
- Конкретные версии инструментов/платформ — только в каталоге версий (`config/toolchain/`), не
  в конфигурации приложений (TVMS).
