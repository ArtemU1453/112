# Reference Information Standard — ЕАСУР

- **Status:** APPROVED · Stage 13 · **Дополняет** Reference Data Management Standard (Stage 9) на уровне
  платформы знаний. Согласован с ADR-028, Master Data Governance Standard, Canonical Identifier Standard.

> Соотношение: Reference Data Management Standard (Stage 9) регулирует НСИ для межведомственного обмена
> (версии/публикация/синхронизация). Данный стандарт регулирует **справочную информацию платформы
> знаний** (классификаторы/словари/коды/перечни/нормативные значения) для диспетчеров/руководителей,
> не дублируя, а специализируя.

## Назначение
Централизованное управление справочной информацией (13.5): классификаторы, словари, коды, перечни,
нормативные значения, отраслевые справочники.

## Принципы
- **Единый источник:** справочники ведутся централизованно; применяются только утверждённые версии.
- **Версионирование:** каждая версия справочника фиксируется; история неизменяема (не удаляется).
- **Каноничность идентификаторов:** согласование с Canonical Identifier Standard (Stage 9).
- **Согласованность:** ссылки правил/SOP/документов используют актуальные коды справочников.
- **Публикация:** изменения — через утверждённый процесс (Knowledge Publication Standard).

## Управление
Владельцы справочников; регламент обновления и публикации; синхронизация с НСИ (Stage 9) при пересечении;
контроль качества (полнота/непротиворечивость, Knowledge Quality Standard).

## Definition of Done
Определены состав, версионирование, каноничность, согласованность и управление справочной информацией.
