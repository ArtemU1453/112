# Knowledge Lifecycle Standard — ЕАСУР

- **Status:** APPROVED · Stage 13 · Согласован с ADR-028, Knowledge Platform Governance Standard,
  Knowledge Retention Standard, Regulatory Document Standard, Knowledge Governance Standard (Stage 11).

## Назначение
Единый жизненный цикл единиц знаний платформы (13.1–13.10): нормативные документы, инструкции, SOP,
методические/учебные материалы, техдокументация, справочники, шаблоны, правила.

## Фазы жизненного цикла
```
Created → Reviewed → Approved → Published → Maintained → Deprecated → Archived
```
- Переходы фиксируются (владелец, дата, основание); публикация — только утверждённых единиц.
- **Deprecated/Archived** — единица помечается статусом и сохраняется; **удаление истории запрещено**.

## Владение и пересмотр
Каждая единица знаний имеет владельца, статус, срок пересмотра. Периодический пересмотр актуальности
(Knowledge Quality Standard); устаревающие выявляются аналитикой (13.10) и актуализируются.

## Трассируемость
Единицы знаний связаны с основаниями (нормативы), зависимостями и версиями; сквозная прослеживаемость
изменений (кто/когда/почему). Соответствие Constitution (документация — SSOT, приоритет над кодом).

## Правила
- Применяются только Published-единицы; Draft/Deprecated в работе не используются.
- Изменение — через утверждённый процесс (Knowledge Publication Standard) и версионирование.
- Архитектурные/процессные изменения по итогам знаний — через RFC→ARB→ADR.

## Definition of Done
Определены фазы, владение, пересмотр, трассируемость и правила жизненного цикла знаний.
