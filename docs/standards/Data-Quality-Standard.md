# Data Quality Standard — ЕАСУР

- **Status:** APPROVED · Stage 8 · Согласован с Data Lineage / Master Data Governance Standards, DGS.

## Контроль качества (8.9)
Поиск дубликатов; контроль полноты; контроль непротиворечивости; контроль актуальности; контроль
происхождения данных.

## Правила
- Правила качества выполняются при загрузке в DWH и периодически; нарушения фиксируются и
  визуализируются (метрики качества).
- Дубликаты выявляются и разрешаются по мастер-данным (Master Data Governance Standard).
- Полнота/непротиворечивость/актуальность измеряются метриками; пороги конфигурируемы.
- Происхождение данных прослеживается (Data Lineage Standard); скрывать происхождение запрещено.
- Некачественные данные помечаются и не искажают отчёты (изоляция/пометка).

## Definition of Done
Определены дубликаты/полнота/непротиворечивость/актуальность/происхождение и обработка нарушений.
