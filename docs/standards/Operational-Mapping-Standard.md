# Operational Mapping Standard — ЕАСУР

- **Status:** APPROVED · Stage 6 · Согласован с GIS Architecture, Map Layer Governance Standard.

## Оперативная карта (6.7)
Отображение: инцидентов; подразделений; маршрутов; зон ответственности. Функции: фильтрация; поиск;
временная шкала событий. Клиент — OpenLayers (ADR-004) поверх self-hosted слоёв (ADR-015).

## Правила
- **Все изменения отражаются на оперативной карте** (обязательная проверка этапа) — обновления в
  реальном времени (realtime-service, Kafka→STOMP).
- Цветовая индикация приоритетов/статусов; кластеризация при плотности; фильтры по типу/статусу/
  службе.
- Доступ к чувствительным слоям — по правам (Spatial Security Standard).

## Definition of Done
Определены отображение инцидентов/подразделений/маршрутов/зон, фильтрация/поиск/временная шкала,
обновление в реальном времени.
