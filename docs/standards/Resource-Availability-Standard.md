# Resource Availability Standard — ЕАСУР

- **Status:** APPROVED · Stage 6 · Согласован с Resource Registry Standard.

## Доступность (6.4)
- Текущее состояние подразделений (`UnitStatus`); загрузка; занятость; резерв; обслуживание
  техники; временная недоступность; прогноз освобождения (ETA возврата/завершения задания).

## Правила
- Доступность вычисляется из реестра (6.3) и трекинга (6.8); учитывается в рекомендациях (6.6).
- Прогноз освобождения — оценочный (на основе статуса/маршрута), не решение; диспетчер учитывает
  при выборе.
- Изменения доступности журналируются и отражаются на оперативной карте.

## Definition of Done
Определены состояние/загрузка/занятость/резерв/обслуживание/недоступность/прогноз освобождения.
