# Mobile Accessibility Standard — ЕАСУР

- **Status:** APPROVED · Stage 7 · Согласован с Mobile Design System.

## Требования (UX)
- **Работа в перчатках:** увеличенные зоны нажатия (≥ 48dp, режим перчаток ≥ 64dp).
- **Крупные элементы управления** и масштабируемый текст (минимальный размер основного текста).
- **Режим высокой контрастности** и **ночной режим** (работа в темноте/ярком свете).
- Поддержка программ чтения с экрана; понятные состояния фокуса; отсутствие зависимости только от цвета.
- Минимизация точности ввода (крупные списки/кнопки, подтверждения критичных действий).

## Правила
Единые компоненты и токены (Mobile Design System); доступность проверяется в тестах UI. Единый дизайн
обязателен во всех приложениях.

## Definition of Done
Определены перчатки/крупные элементы/контраст/темы/скринридер/минимизация точности ввода.
