# Coordinate Reference System Standard — ЕАСУР

- **Status:** APPROVED · Stage 6 · Согласован с Geospatial Data Model, ADR-003.

## Правила
- **Обмен и хранение координат** — WGS84 (EPSG:4326), порядок lat/lon; координаты — WGS84 всюду в
  контрактах API (Emergency Data Dictionary §6).
- **Метрические расчёты** (расстояние, буфер, площадь) — метрическая проекция (UTM для территории
  Беларуси) или геодезические функции PostGIS (`ST_Distance` с географией).
- Преобразования CRS — только на границах (адаптеры/сервис GIS), не в прикладной логике.
- Точность координат фиксируется (не менее 5–6 знаков после запятой).

## Обязательность
Все пространственные объекты и контракты используют единый CRS обмена. Отклонения — через RFC/ADR.

## Definition of Done
Определён CRS обмена/хранения и правила проекций/преобразований; согласовано с моделью данных.
