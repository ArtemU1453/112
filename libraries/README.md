# libraries/ — общие библиотеки и сборочные артефакты

Стабильные, не-доменные кросс-срезы и конфигурация сборки. Доменный код между сервисами не
разделяется (автономность bounded contexts выше DRY между ними — см. System Structure).

- `build/parent` — родительская Maven-конфигурация (Java, управление плагинами).
- `build/bom` — Bill of Materials (централизованное управление версиями библиотек, DepGS).
