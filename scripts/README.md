# scripts/ — автоматизация разработки

Единые скрипты для воспроизводимых операций (Repository Automation, Developer Experience).

| Скрипт | Назначение |
|--------|-----------|
| bootstrap.sh | Проверки + запуск среды разработки (docker/compose.dev) |
| build.sh | Сборка всех компонентов |
| test.sh | Прогон тестов |
| lint.sh | Линтеры/анализаторы (config/quality) |
| verify.sh | Локальная валидация архитектуры/версий/документации |
| verify-versions.sh | Соответствие инструментов каталогу версий |
| verify-architecture.sh | Проверка структуры репозитория и правил |
