# docker/ — окружения на Docker Compose

Compose-файлы для разработки и вспомогательных задач (Environment Governance Standard).
Секреты не хранятся здесь — только плейсхолдеры/переменные окружения.

| Файл | Назначение |
|------|-----------|
| compose.dev.yml | Инфраструктура для локальной разработки (БД, брокер, кэш, IdP) |
| compose.test.yml | Изолированная инфраструктура для интеграционных тестов |
| compose.tools.yml | Инструменты разработчика (линтеры/сканеры) |
| compose.monitoring.yml | Локальный стек наблюдаемости |

Полный запуск системы целиком — корневой `../docker-compose.yml`.
