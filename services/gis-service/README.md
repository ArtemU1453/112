# gis-service

Геоинформационный сервис системы 112 на PostGIS.

- Порт: **8085**, БД: `gis_db` (PostGIS + pg_trgm)
- Прямое геокодирование (адрес→координаты) через нечёткий поиск pg_trgm
- Обратное геокодирование (координаты→ближайший адрес, KNN `<->`)
- Ближайшие станции службы (`ST_Distance` по географии, KNN-сортировка)
- Определение зон ответственности точки (`ST_Contains`)
- Предзаполненный справочник: пожарные части и адреса Минска (Liquibase seed)

## Тесты
