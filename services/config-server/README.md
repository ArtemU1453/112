# config-server (плейсхолдер)

**Статус:** планируется (не реализовано). Каталог-плейсхолдер (Stage 2 структура monorepo).

- **Назначение:** централизованная выдача конфигурации сервисам (возможность «config server»).
- **Примечание:** на текущей Baseline конфигурация инжектируется через переменные окружения
  (Configuration Standards). Введение отдельного config-server — только через ADR/RFC.
